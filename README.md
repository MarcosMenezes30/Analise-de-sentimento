# Sistema de Classificação de Sentimento em Português com NLP e Machine Learning

O projeto simula um sistema de análise automática de feedback de clientes, classificando textos em **positivo**, **negativo** e **neutro** usando **scikit-learn** para apoiar tomada de decisão.

Projeto para o Clube de Inteligência Artificial e Ciência de Dados do Senai Cimatec.

Projeto criado em Novembro de 2025 e publicado no GitHub em Abril de 2026.

## Como rodar

```bash
pip install -r requirements.txt
python train.py
streamlit run app.py
